## 🔨 <b>Projects</b>

<p>
<b>Poincaré</b> — v0.5.3-beta, in active development
<br>
A C++ CLI application and mathematical library, organized into a proper <code>UI::header</code> / <code>UI::body</code> namespace structure. Current modules:
</p>

<ul>
<li><b>Geometry</b> — circle, square, rectangle, 2D shapes (polygons, quadrilaterals, triangles, regular polygons), conic section classification from a general equation (circle / ellipse / parabola / hyperbola), and 3D shapes (cube, rectangular prism, sphere, cylinder, cone, square pyramid)</li>
<li><b>Unit Converters</b> — angle, length, area, volume, mass, temperature, speed, time, pressure</li>
<li><b>Matrices</b> — dynamic-size operations up to 8×8. Single-matrix tools (transpose, inverse, scalar multiplication, integer power, trace, determinant) plus two-matrix operations (sum, difference, product, trace)</li>
<li><b>Linear System Solver</b> — solves systems up to 26 equations via Gaussian elimination with partial pivoting, with singular-system detection</li>
<li><b>Polynomial Equations</b> — closed-form solving for linear and quadratic, plus numerical root-finding (Durand–Kerner) up to degree 26, with complex coefficients</li>
<li><b>Complex Numbers</b> — full arithmetic, roots & powers, exponential/logarithmic, trigonometric & hyperbolic functions, and boolean properties — for a single number, a pair, or an arbitrary set</li>
<li><b>Number Theory</b> — constants, function application over sequences, arithmetic/geometric/Fibonacci/prime sequences, prime factorization, and set operations (sum, product, parity, primality)</li>
</ul>

<p>
Built on top of custom headers I'm developing as a reusable math library.
<br>
Repository: <a href="https://github.com/AnasRiemann/Poincare_CLI_app">Poincare_CLI_app</a>
<br>
More coming soon as I get things upload-ready.
</p>
